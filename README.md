## Descrição

O "MyWallet" é um projeto completo de acompanhamento de gastos e receitas, projetado para ajudar os usuários a acompanharem suas contas.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando:

- **React**
- **Node.js**
- **MongoDB**
